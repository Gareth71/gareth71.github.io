Public and Private detached rule sets

A detached ruleset is a group of css properties, nested rulesets, media declarations or anything else stored in a variable. On their own they don’t appear in the CSS until they are called by adding a () at the end.

Public rule sets begin with @ and are for child design packs to be able to push styles up into a parent.

Private ones begin with @_ and are for parent design packs to apply styles to selectors in their own or child design packs along with the contents of the public rule set. Avallain state Private rule sets should never be modified.

Avallains design pack is built around creating a public and private detached rule set where ever possible for each set of rules and apply them to small amount of selectors. Child packs then update the public rule set and dont create new selectors unless necessary. This should result in only a few selectors applying to any given element in the DOM.

Example of Avallains public and private detached rule set pattern

/* Create a public version of the rule set */
@detached-rule-set: {}

/* Create a private version with rules that should not be removed */
@_detached-rule-set: {
  color: red;
  padding: 1rem;
}

/* Add private ruleset to a selector (add opening and closing brackets at the end */
body {
  @_detached-rule-set()
}

/* compiled result */
body {
  color: red;
  padding: 1rem;
}

A child design pack can add additional CSS to the public rule set and it will appear inside the private ruleset in the parent design pack. No need to apply to a selector. 

@detached-rule-set: {
  color: green;
}

/* compiled result *
body {
  color: red;
  padding: 1rem;
  color: green;
}

If the public ruleset is declared multiple times the last one to be declared overrides the previous ones. This will be important in design packs with grandparents as they could override a public rules set in their parent.

@detached-rule-set: {
  border: 1px solid red;
}

@detached-rule-set: {
  color: green;
}

/* compiled result *
body {
  color: red;
  padding: 1rem;
  color: green;
}

/* Detached ruleset PUBLIC (@) */
@icCrossword__grid: {};

/* Detached rulest PRIVARE (@_) DONT EDIT!*/
@_icCrossword__grid: {
	position: relative;
	margin: 0 auto;
	/* Public details are added to private */
	@icCrossword__grid();
};

/* PRIVATE Detached rulest is added to selector */
.ic-crossword {
	@_icCrossword__grid();
}

/*  child design pack 1 */
@icCrossword__grid: {
	margin: 20px;
};

/*  child design pack 2 */
@icCrossword__grid: {
	margin: 40px;
};

/* Overide the Top level ruleset */
@_icCrossword__grid: {
	@icCrossword__grid();
}
