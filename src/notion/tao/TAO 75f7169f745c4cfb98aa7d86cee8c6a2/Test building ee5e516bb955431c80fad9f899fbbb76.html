Export

Edit

  1. Unzip the export

test.xml

  1. modify identifier and title so that you can identify the test when you import it back into TAO, I add " _v1" to the end of both and an underscore at the start of the title.

    The Title seems to be the part that used the most in TAO for the imported items and the test name

<?xml version="1.0" encoding="UTF-8"?>
<assessmentTest xmlns="http://www.imsglobal.org/xsd/imsqti_v2p1"
xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
identifier="_Primo_2_v1"
title="_Primo 2 v1"
toolName="tao"
toolVersion="3.2.0-RC2"
xsi:schemaLocation="http://www.imsglobal.org/xsd/imsqti_v2p1 http://www.imsglobal.org/xsd/qti/qtiv2p1/imsqti_v2p1.xsd">
  1. Replace the top three outcomeDeclaration with the following to the top
<outcomeDeclaration identifier="SCORE_TOTAL" cardinality="single" baseType="float"/>
  <outcomeDeclaration identifier="SCORE_CATEGORY_READING" cardinality="single" baseType="float"/>
  <outcomeDeclaration identifier="SCORE_CATEGORY_LISTENING" cardinality="single" baseType="float"/>
<outcomeDeclaration identifier="SCORE_TOTAL_MAX" cardinality="single" baseType="float"/>
  <outcomeDeclaration identifier="SCORE_CATEGORY_MAX_READING" cardinality="single" baseType="float"/>
  <outcomeDeclaration identifier="SCORE_CATEGORY_MAX_LISTENING" cardinality="single" baseType="float"/>
<outcomeDeclaration identifier="SCORE_RATIO" cardinality="single" baseType="float"/>
  1. Not required if the test already has had the results page added
    insert the following after <timeLimits allowLateSubmission="false"/> in the last assessmentSection and adjust the red scores and test title
<rubricBlock view="candidate">
        You have just finished
        <h1>Secundo Grado: Ready for Invalsi 5a Online Test 1</h1>
        <h2>You scored: <printedVariable identifier="SCORE_TOTAL" base="10" powerForm="false" delimiter=";" mappingIndicator="=" format="%d"/>/88</h2>
        <ul class="results-list">
          <li>
            Reading:
            <printedVariable identifier="SCORE_CATEGORY_READING" base="10" powerForm="false" delimiter=";" mappingIndicator="=" format="%d"/>/42
          </li>
          <li>
            Listening:
            <printedVariable identifier="SCORE_CATEGORY_LISTENING" base="10" powerForm="false" delimiter=";" mappingIndicator="=" format="%d"/>/46
          </li>
        </ul>
</rubricBlock>
  1. Replace outcomeProcessing at the very end with this.
<outcomeProcessing>
    <setOutcomeValue identifier="SCORE_TOTAL">
      <sum>
        <testVariables variableIdentifier="SCORE"/>
      </sum>
    </setOutcomeValue>
    <setOutcomeValue identifier="SCORE_CATEGORY_READING">
      <sum>
        <testVariables includeCategory="Reading" variableIdentifier="SCORE"/>
      </sum>
    </setOutcomeValue>
    <setOutcomeValue identifier="SCORE_CATEGORY_LISTENING">
      <sum>
        <testVariables includeCategory="Listening" variableIdentifier="SCORE"/>
      </sum>
    </setOutcomeValue>
    <setOutcomeValue identifier="SCORE_TOTAL_MAX">
      <sum>
        <testVariables variableIdentifier="MAXSCORE"/>
      </sum>
    </setOutcomeValue>
    <setOutcomeValue identifier="SCORE_CATEGORY_MAX_READING">
      <sum>
        <testVariables includeCategory="Reading" variableIdentifier="MAXSCORE"/>
      </sum>
    </setOutcomeValue>
    <setOutcomeValue identifier="SCORE_CATEGORY_MAX_LISTENING">
      <sum>
        <testVariables includeCategory="Listening" variableIdentifier="MAXSCORE"/>
      </sum>
    </setOutcomeValue>
    <outcomeCondition>
      <outcomeIf>
        <isNull>
          <variable identifier="SCORE_TOTAL_MAX"/>
        </isNull>
        <setOutcomeValue identifier="SCORE_RATIO">
          <baseValue baseType="float">0</baseValue>
        </setOutcomeValue>
      </outcomeIf>
      <outcomeElse>
        <setOutcomeValue identifier="SCORE_RATIO">
          <divide>
            <variable identifier="SCORE_TOTAL"/>
            <variable identifier="SCORE_TOTAL_MAX"/>
          </divide>
        </setOutcomeValue>
      </outcomeElse>
    </outcomeCondition>
  </outcomeProcessing>

Results

Re-import

Select the xml file and the two folders and choose Compress 3 items from the contextual menu.

The resulting Archive is what you import to TAO.

Correct the test name to its full version.

Once it has been imported you can assign a category to each section. All items inside it will be given the category.
