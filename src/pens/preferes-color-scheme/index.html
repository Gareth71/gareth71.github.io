Prefers color scheme

Prefers-color-scheme Media queries prefers

User color scheme preferences set at device or browser level can be accessed via the media query "prefers-color-scheme"

Links can be conditionally loaded like this

<link rel="stylesheet" href="index.css" media="(prefers-color-scheme: dark)" /> <link rel="icon" href="/white-logo.png" media="(prefers-color-scheme: dark)" />
