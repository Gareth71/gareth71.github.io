Heading line-height

Using calc to figure out optimal line height

The bigger type gets the tighter its line-height should be. A line-height of 1.1 works for very large type but is too tight for smaller headings. Instead of specifying line-heights for each heading size we can calculate it like this. 

h1,h2,h3,h4,h5,h6 {
    line-height: calc(0.125rem + 1em + 0.125rem);
}

With a 24px font with 1.5 line-height, the text takes up 1em (24px) and the remaining space (24px) is split above and below.

With the calc method we only add 2px above and below. This is always the same amount of space between lines regardless of font size.
