SVG sprites

Complete guide to SVG sprites

All your svgs can be cached inside an svg sprite sheet and accessed without any additional loading.

Create the sprite sheet like this

<?xml version="1.0" encoding="utf-8"?> <svg xmlns="http://www.w3.org/2000/svg" style="display: none;"> <defs> <!-- icons go here --> </defs> </svg>

Paste svgs into it, change svg to symbol and add an id for each one.

<?xml version="1.0" encoding="utf-8"?> <svg xmlns="http://www.w3.org/2000/svg" style="display: none;"> <defs> <symbol viewBox="0 0 100 100" fill="none" xmlns="http://www.w3.org/2000/svg" id="notion"> <path d="M6.017 4.313l55.333 -4.087c6.797 -0.583 8.543 -0.19 ..." fill="#fff"> <path fill-rule="evenodd" clip-rule="evenodd" d="M61.35 ..." fill="#000"/> </symbol> </defs> </svg>

When you want to use a svg insert the use element like this with the id of the symbol.

<svg class="icon"> <use href="#notion" aria-hidden="true"> </svg>
