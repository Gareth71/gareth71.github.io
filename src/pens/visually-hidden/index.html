Visually hidden

You can visually hide items but leave them accessible to assisted technology by adding a class with these styles. This is better than applying aria-label or alt text, as text in the DOM is more likely to be translated than text inside any form of attribute.

.visually-hidden:not(:focus):not(:active) { border: 0 !important; clip: rect(0 0 0 0) !important; clip-path: inset(50%) !important; height: 1px !important; margin: -1px !important; overflow: hidden !important; padding: 0 !important; position: absolute !important; white-space: nowrap !important; width: 1px !important; }
  • :not(:focus):not(:active) is because anything focused or active should be visible to the user.
  • !important is to avoid and other styles leaking in.
  • For full details see The anatomy of visually hidden

The class can now be used inside say a button that only has an icon

<button> <svg version="1.1" width="24" height="24" xmlns="http://www.w3.org/2000/svg"> <rect width="100%" height="100%" fill="red" /> </svg> <span class="visually-hidden">Close</span> </button>
